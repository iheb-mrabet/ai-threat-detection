# Project Summary

This project is a real-time AI threat detection platform for web traffic.

It monitors nginx-style access logs, extracts features, detects malicious requests using rules and ML models, stores events in PostgreSQL, streams alerts through Redis/WebSocket, and visualizes results in a React dashboard.

The system also includes security evaluation metrics such as FAR, FRR, EER, ROC-AUC, PR-AUC, and a confusion matrix.
